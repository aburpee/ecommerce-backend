# E-commerce Back End

## Description
This back-end api allows database management for the following tables:
* Products
* Categories
* Tags

The tables allow the following requests to fullfil CRUD requirements:
* get
* post
* put
* delete

## Technologies
* express
* sequelize
* mysql2
* javascript
